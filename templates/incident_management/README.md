# Incident Template Options

This folder contains ready-to-use options for incident documentation and compliance operations.

## 1) Printable Form
- File: incident_form_printable.md
- Best for: manual entry, interviews, and formal sign-off packets.

## 2) Spreadsheet Template
- File: incident_template.csv
- Best for: tracking many incidents and reporting trends.
- Includes: one sample row and all workflow fields.

## 3) Dropdown Reference
- File: dropdown_reference.csv
- Best for: applying data validation lists in Excel or Google Sheets.

## 4) Policy Appendix
- File: policy_appendix_incident_workflow.md
- Best for: governance documents, SOPs, and audit evidence.

## Important Control Rules
- One primary domain per incident.
- One owning department at a time.
- Other departments are advisory only.
- Public information requirement is tracked with a 30-day deadline field.
